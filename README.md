# CMC Whale Skill - Quantitative Strategy Analyzer

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![TypeScript](https://img.shields.io/badge/typescript-100%25-blue)

A CLI-based autonomous quantitative trading strategy analyzer built for the **BNB Chain × CoinMarketCap × Trust Wallet AI Agent Hackathon**. 

This application fetches real-time cryptocurrency market data (price, 24h volume, 24h percent change) directly via the CoinMarketCap API and injects it into a Google Gemini AI model. The AI acts as a quantitative strategist, parsing the live market anomalies and outputting a highly professional, backtestable trading strategy specification (Track 2: Strategy Skills).

## Features
- **Real-Time Market Data**: Integration with CoinMarketCap's Pro API to fetch the top cryptocurrencies by market capitalization.
- **AI-Powered Analytics**: Uses Google's Gemini LLM to interpret market conditions and formulate algorithmic trading rules.
- **Clean Architecture**: Built with TypeScript using a clean, scalable architectural pattern separating Infrastructure (API Clients) from the Application Layer (Use Cases).
- **Zero-Cost Operation**: Designed to run entirely on the free tiers of CoinMarketCap API and Google AI Studio.

## Architecture

```text
src/
 ├── config/
 │    └── env.ts                 # Environment variables & configuration
 ├── infrastructure/
 │    ├── cmcClient.ts           # CoinMarketCap API integration
 │    └── geminiClient.ts        # Google Gemini AI integration
 ├── application/
 │    └── analyzerUseCase.ts     # Core business logic: Data fetch -> AI prompt -> Output
 └── index.ts                    # Clean application entry point
```

## Prerequisites
- **Node.js** (v18 or higher)
- **npm** or **yarn**
- **CoinMarketCap API Key**: Get it from [CoinMarketCap Developer Portal](https://pro.coinmarketcap.com/)
- **Gemini API Key**: Get it from [Google AI Studio](https://aistudio.google.com/app/apikey)

## Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/ekosanto2008/cmc-whale-skill.git
   cd cmc-whale-skill
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Environment Setup**
   Copy the example environment file and add your actual API keys.
   ```bash
   cp .env.example .env
   ```
   Open `.env` and fill in:
   ```env
   CMC_API_KEY=your_cmc_api_key_here
   GEMINI_API_KEY=your_gemini_api_key_here
   ```

## Usage

Run the analyzer via the following command:

```bash
npm run start
```

### Expected Output
The application will print the fetched tokens and then output a structured **Quantitative Strategy Specification** generated by the AI, which includes:
- Market Overview
- Selected Coins for Trading (based on current volatility/liquidity)
- Algorithmic Entry Conditions
- Algorithmic Exit Conditions (Stop Loss / Take Profit)
- Edge / Rationale
